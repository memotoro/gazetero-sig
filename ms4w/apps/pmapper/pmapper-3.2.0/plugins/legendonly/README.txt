This is a small plugin for pmapper framework.
It just hide toc in order to show only the legend.

It contains:
- config.inc: include the other files in pmapper
- legendonly.js: just the function to call after TOC loading

Dependancies :
No dependancy

How to use:
Add the string "legendonly" to the plugins list in the pmapper config file.
Be carreful, this plugin will force legStyle in swap mode.

Parameters:
No parameters.
